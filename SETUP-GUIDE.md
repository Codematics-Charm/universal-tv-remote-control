# Setup Guide for Universal TV Remote Control

This guide explains how to set up Universal TV Remote Control by Codematics with your TV.

## WiFi Setup

1. Connect your phone and Smart TV to the same WiFi network.
2. Open Universal TV Remote Control.
3. Select your TV brand or allow the app to search for available TVs.
4. Choose your TV from the list.
5. Accept the pairing request on your TV screen if required.
6. Start using the remote control features.

## IR Setup

1. Make sure your phone has an IR blaster.
2. Open Universal TV Remote Control.
3. Select IR remote mode.
4. Choose your TV brand.
5. Test the power or volume button.
6. Save the remote once it works.

## Important Note

WiFi remote control requires both devices to be connected to the same network. IR remote control requires a phone with built-in IR support.
